Quick start checklist (daily/operator use)
1) Verify external storage is mounted to /mnt/quant-data
   - findmnt /mnt/quant-data
2) Activate virtual environment
   - source .venv/bin/activate
3) Verify Ollama model availability
   - ollama list
   - confirm llama3.1:8b is present
4) Ingest fresh market data (example)
   - quant-agents ingest --exchange kraken --symbol BTC/USDT --timeframe 1h --limit 300
5) Run agent plane with stable timeout (example)
   - OLLAMA_TIMEOUT_SECONDS=180 quant-agents agent-plane --exchange kraken --symbol BTC/USDT --timeframe 1h --strategy-model llama3.1:8b --ops-model llama3.1:8b
6) Generate visual diagnostics
   - quant-agents visualize-run
7) Review outputs in the latest run directory
   - strategy_proposal_signal.json
   - backtest_evaluation.json
   - risk_decision.json
   - paper_trade_intent.json
   - visuals/*.png
