# Quant Trigger Dashboard (Next.js)
This app visualizes prediction and alert artifacts generated by the Python quant pipeline.

## Run locally
```bash
npm install
QUANT_DATA_ROOT=/mnt/quant-data npm run dev
```

## Data sources
- Predictions: `logs/agents/model-predictor/<yyyy-mm-dd>/prediction_*.json`
- Alerts: `logs/agents/trigger-monitor/<yyyy-mm-dd>/alerts.jsonl`
- Optional context: latest `logs/agents/openclaw-orchestrator/**/run_manifest.json`

The app exposes these through `GET /api/overview` and renders an interactive UI in `app/page.tsx`.
